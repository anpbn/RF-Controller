# 433 RF 编辑器

ESP8266 + 433MHz 射频发射模块的自定义信号编辑器。

## 功能

- **编辑信号**：自定义十进制代码、协议类型、位数
- **发送一次**：立即发射单次信号
- **连续发送**：按设定间隔重复发射，随时停止
- **定时发送**：延迟指定秒后发射一次

## 硬件接线

| RF模块 | ESP8266 |
|--------|---------|
| VCC    | 3.3V    |
| GND    | GND     |
| DATA   | D6 (GPIO12) |

## 使用

1. 手机连接 WiFi `RF-Editor`，密码 `12345678`
2. 浏览器访问 `192.168.4.1`
3. 输入信号参数，点击对应按钮
